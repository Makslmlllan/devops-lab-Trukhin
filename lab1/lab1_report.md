# Лабораторная работа №1
Основы работы с Docker

Цель работы
Изучить основы работы с Docker: создание контейнеров, управление образами и использование томов.

Ход работы

1. Установлен Docker Desktop.
2. Проверена установка командой docker --version.
3. Запущен тестовый контейнер hello-world.
4. Загружен образ Ubuntu.
5. Создан и запущен контейнер Ubuntu.
6. Внутри контейнера установлен пакет curl.
7. Запущен веб-сервер nginx и проверена его работа.
8. Изучены команды управления контейнерами.
9. Создан Docker volume и проверено сохранение данных.

Результат
В ходе работы были изучены основные команды Docker, запуск контейнеров, управление образами и работа с томами.
